# AptS-1547/homebrew-tap

Homebrew Tap for AptS-1547 projects.

## Usage

```bash
brew tap AptS-1547/tap
```

## Formulae

| Name | Description | Install |
|------|-------------|---------|
| [gcop-rs](https://github.com/AptS-1547/gcop-rs) | AI-powered Git commit message generator and code reviewer | `brew install AptS-1547/tap/gcop-rs` |
| [shortlinker](https://github.com/AptS-1547/shortlinker) | A minimalist URL shortener service supporting HTTP 307 redirection | `brew install AptS-1547/tap/shortlinker` |

## Casks

| Name | Description | Install |
|------|-------------|---------|
| [dns-orchestrator](https://github.com/AptS-1547/dns-orchestrator) | DNS configuration management tool | `brew install --cask AptS-1547/tap/dns-orchestrator` |
