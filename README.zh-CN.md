# AptS-1547/homebrew-tap

AptS-1547 项目的 Homebrew Tap 仓库。

## 使用方法

```bash
brew tap AptS-1547/tap
```

## 命令行工具（Formulae）

| 名称 | 描述 | 安装命令 |
|------|------|----------|
| [gcop-rs](https://github.com/AptS-1547/gcop-rs) | AI 驱动的 Git 提交信息生成器与代码审查工具 | `brew install AptS-1547/tap/gcop-rs` |
| [shortlinker](https://github.com/AptS-1547/shortlinker) | 极简 URL 短链接服务，支持 HTTP 307 重定向 | `brew install AptS-1547/tap/shortlinker` |

## 桌面应用（Casks）

| 名称 | 描述 | 安装命令 |
|------|------|----------|
| [dns-orchestrator](https://github.com/AptS-1547/dns-orchestrator) | DNS 配置管理工具 | `brew install --cask AptS-1547/tap/dns-orchestrator` |
